# Política de Seguridad de b24cloud

## 🔐 Reporte de Vulnerabilidades

En b24cloud, la seguridad es una prioridad. Si encuentras una vulnerabilidad de seguridad en alguno de nuestros proyectos, **te agradecemos que nos lo reportes de manera responsable** para que podamos abordarlo de inmediato.

### 📌 Cómo reportar una vulnerabilidad

1. **Envía un correo electrónico** a [security@b24cloud.com](mailto:security@b24cloud.com) con los detalles de la vulnerabilidad.
   - Incluye una descripción clara del problema.
   - Si es posible, proporciona pasos para reproducirlo.
   - Añade cualquier información relevante (versiones, entornos, etc.).

2. **Espera nuestra respuesta** dentro de las **72 horas**. Trabajaremos para validar y solucionar el problema lo antes posible.

3. **No publiques detalles públicos** de la vulnerabilidad hasta que hayamos solucionado el problema y te hayamos notificado.

### 🚫 ¿Qué NO hacer?

- No crees issues públicos en los repositorios para reportar vulnerabilidades.
- No compartas información sensible (tokens, contraseñas, etc.) en foros públicos.
- No intentes explotar la vulnerabilidad más allá de lo necesario para demostrarla.

### 📜 Compromiso

- **Confidencialidad:** Tu reporte se manejará con la máxima confidencialidad.
- **Reconocimiento:** Si lo deseas, podemos reconocer tu contribución públicamente (opcional).
- **Recompensas:** En algunos casos, podemos ofrecer recompensas por vulnerabilidades críticas (dependiendo del proyecto).

### 🔍 Ejemplos de vulnerabilidades comunes

- Inyección de código (SQL, Command Injection, etc.).
- Cross-Site Scripting (XSS).
- Cross-Site Request Forgery (CSRF).
- Exposición de datos sensibles (API keys, tokens, etc.).
- Configuraciones inseguras (CORS, permisos, etc.).

### 📅 Proceso de solución

1. **Validación:** Confirmamos la vulnerabilidad.
2. **Corrección:** Desarrollamos y probamos un parche.
3. **Notificación:** Te informamos sobre la solución y el cronograma de despliegue.
4. **Publicación:** Si es relevante, publicamos un aviso de seguridad (CVE, advisory, etc.).

### 📞 Contacto

Si tienes dudas sobre seguridad, escríbenos a [security@b24cloud.com](mailto:security@b24cloud.com).
